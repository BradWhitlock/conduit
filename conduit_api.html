Generation from JSON Schemas

void generate(const Generator &gen)¶

description: These methods use a Generator to parse a json schema into a Node hierarchy.

  • The variants without a void * data parameter will allocate memory for the Node hierarchy and populate with inline values from the json schema (if they are provided).
  • The `external’ variants build a Node hierarchy that points to the input data, they do not copy the data into the Node hierarchy.

bool is_contiguous() const¶

contiguous checks

A node is contiguous if the leaves of it children (traversed in a depth first order) cover a contiguous chunk of the address space.

The direct address checks are only done for leaves with data, nodes in the objects, lists, or empty roles don’t directly advance the pointer.

Checks use each leaf’s offset and the total strided bytes If leaves do not abut in address space, or if any leaf points to NULL the Node is not contiguous.

This check is agnostic to if the Node owns the data. Does this node has a contiguous data layout?
